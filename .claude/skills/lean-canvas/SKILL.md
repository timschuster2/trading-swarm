Frame or stress-test a product/service idea before committing to a full build.

## When to Use
- "Is this worth building?"
- "What should this be?"
- Validating a new product or service idea
- Before investing in requirements or design

## Lean Canvas Template

```
LEAN CANVAS — [Product/Service] — [Date]

PROBLEM:           [Top 3 problems]
CUSTOMER SEGMENTS: [Target users]
UNIQUE VALUE PROP: [Single clear sentence]
SOLUTION:          [Top 3 features]
CHANNELS:          [How you reach customers]
REVENUE STREAMS:   [How you make money]
COST STRUCTURE:    [Key costs]
KEY METRICS:       [Numbers that matter]
UNFAIR ADVANTAGE:  [What can't be easily copied]
```

## Output

```
LEAN CANVAS — [Product] — v[N] — [Date]
Status: VIABLE / PIVOT / PARK
Key risk: [single biggest unknown]
Next: [requirements / park / research needed]
```
